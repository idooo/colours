Colours

Touch friendly color picker for people. For tasks where users do not want to select exactly #26466D color,
but just want to find similar color to pick. Do not make them think!

Text color

    $('#control2').colours({
        palette: flat_palette,
        color: 'black',
        width: 200,
        show_field: true,
        palette_size: 26,
        palette_row_count: 8,
        onColorChange: function(color) {
            $('body').css('color', color);
        }
    });

Callbacks and helpers to save your day

Yet another picker
#95a5a6

    $('#control3').colours({
        palette: flat_palette,
        color: '#95a5a6',
        onColorChange: function(color) {
            $('#example')
                    .text(color)
                    .css('background-color', color);
        },
        onPaletteOver: function(color) {
            $('#example').css('background-color', color);
        },
        onPaletteOut: function(color) {
            $('#example').css('background-color', this.color);
        }
    });
